# CLAUDE.md - AI 에이전트 필수 규칙

> 이 파일은 모든 작업 전에 자동으로 읽힌다.
> 여기에 적힌 규칙은 선택이 아니다 **필수**다.

---

## 1단계: 당일 LOG 파일 생성

- 일자별로 logs 폴더에 LOG 파일을 생성한다. LOG 파일 명칭 포맷은 yyyy-mm-dd
- 각 로그 파일은 해당 년이 지나면 logs 폴더 하위에 xxxx년 폴더 생성 후 포함되는 폴더, 파일들 이동
- 각 로그 파일은 해당 월이 지나면 logs 폴더 하위 xxxx년 폴더 하위에 xx월 생성 후 하위 폴더, 파일들 이동
- 모든 실행 명령과 종료 결과, 오류가 발생했다면 오류 내용을 로그 파일에 저장한다.
- 로그 내용 포맷은 모든 내용 맨 앞에 현재 한국 시간을 "[xxxx-mm-dd hh:mm:ss] 로그 내용" 형태로 입력한다.

## 2단계: PLAN 생성

- 모든 작업은 PLAN 생성 후 시작한다.
- 생성한 PLAN 내용은 로그 파일에 저장한다.
- PLAN 에 없는 내용은 절대 작업하지 않는다.
